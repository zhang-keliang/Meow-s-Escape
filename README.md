## 🎮 Introduction (项目简介)

**Meow's Escape** 是一款基于经典“贪吃蛇”机制的生存恐怖游戏，但加入了紧张刺激的**追逐战**元素。

在这个版本中，你不再是悠闲的捕食者。你扮演一只无助的小猫，不仅要在这个充满**剧毒河豚**的世界中寻找食物，还要时刻提防身后那只**变异巨鼠**的疯狂追杀！

这是一个 **Vibe Coding** 实验项目：所有代码逻辑均通过 AI (LLM) 对话生成，未手动编写一行 JavaScript。

## ✨ Key Features (核心玩法)

* 🐱 **主角 (The Cat)**: 使用方向键控制，吃到食物身体变长。
* 🐭 **宿敌 (The Mutant Rat)**: **动态威胁！** 一只不受墙壁限制的巨鼠会实时追踪你的位置。停下即死！
* 🐡 **陷阱 (The Pufferfish)**: **静态威胁！** 地图上随机分布着剧毒河豚，触碰立即游戏结束。
* 🥓 **策略 (Risk & Reward)**:
    * 🌿 **猫草 (Cat Grass)**: +10分，身体 +1。
    * 🥓 **猫条 (Tasty Treat)**: +20分，身体 +2。高分往往伴随着高风险！

## 🕹️ How to Play (操作指南)

1.  **移动**: 使用键盘 `↑` `↓` `←` `→` 控制小猫移动。
2.  **目标**: 尽可能多地吃掉猫草和猫条来获得高分。
3.  **生存规则**:
    * 不要撞墙。
    * 不要撞到自己的身体。
    * **不要碰到河豚 🐡**。
    * **不要被老鼠 🐭 抓到**。
